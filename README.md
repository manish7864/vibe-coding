# vibe-coding

This repository contains small demos. The primary project here is `snake-game-chatgpt`, a Snake game built with React and Vite.

To run the Snake demo locally:

1. Change into the project folder:

   cd snake-game-chatgpt

2. Install dependencies:

   npm install

3. Start the development server:

   npm run dev

4. Open the app in your browser (usually at `http://localhost:5173`), or follow the URL printed in the terminal.

Enjoy!
